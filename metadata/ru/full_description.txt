Блокирует нежелательные звонки и SMS, не заменяя приложение для звонков/SMS по умолчанию.

Фильтры:

1. Правила по регулярному выражению для номера и содержимого SMS.
2. Контакты
3. Подтверждение STIR
4. Повторный звонок
5. Набранный номер
6. Недавно использованные приложения
7. Расписание

Подробности: https://github.com/aj3423/SpamBlocker
